# Contributing to NEXUM Protocol

NEXUM is built by the SEA community, for the SEA community.

## Pull Requests
1. Fork and branch: `git checkout -b feat/your-feature`
2. Make changes with descriptive commits
3. Add tests where applicable
4. Open a PR with clear description

## Code Style
- Rust: `cargo fmt`
- TypeScript: ESLint + Prettier
- Commits: Conventional Commits format

## Community
All contributors from Southeast Asia especially welcome.
Questions? Open a discussion or join our Discord.
